## Headline findings

1. **H2 cross-prompt generalization confirmed at n=480**: paired-delta `PMR(ball) − PMR(circle)` is **positive in all 5 models × 3 prompts = 15 cells**, range +0.006 to +0.344.
2. **0/2160 unparseable on `meta_phys_yesno`** — yes/no parser is reliable across all 5 architectures.
3. **Phase 1↔2 consistency**: stratified n=48 subset numbers are within ±0.018 PMR of full n=480 in all (model × prompt) cells. Stratified subsampling was unbiased.
4. **Saturation × prompt interaction**: for **saturated models** (Idefics2 / Qwen / InternVL3), `describe_scene` reveals the strongest H2 (+0.175 to +0.267); for **unsaturated CLIP-cluster models** (LLaVA-1.5 / LLaVA-Next), `open` reveals the strongest H2 (+0.344 / +0.100). The describe-prompt has more H2 headroom than `open` exactly where `open` saturates label effects.
5. **`meta_phys_yesno` is the most discriminating ladder**: spread 0.32–0.98 vs `open` PMR's 0.67–0.99. Wider headroom for cross-model PMR comparison.

## Acceptance criteria status (per `docs/m_mp_multi_prompt_design.md` §6.2)

- **Strong (best case)**: same `v_L` direction flips across 3 prompts × ≥3 models → mechanism is task-agnostic. **Currently behavioral-only**. Phase 3 (cross-prompt M5a + M5b on Qwen + Idefics2) tests the **causal** version of this claim — week 3.
- **Mixed**: 2 of 3 prompts work → claim refined.
- **Weak (pivot)**: only kinetic prediction works → claim narrows.

Phase 2 result is consistent with **Strong** at the behavioral level — H2 holds in all 15 (model × prompt) cells. Phase 3 will check whether the same `v_L10` direction / SAE features causally drive the H2 across prompts.

## Caveats / limitations

- **`describe_scene` scoring** has a passing Claude-rater hand-label gate (agreement 0.88–1.00, κ 0.74–1.00, all 5 models PASS) — but this is one programmatic rater. Paper writeup should add a 2nd-rater (user) pass for cross-rater confirmation.
- Phase 2 is **behavioral evidence only**. Causal claim ("same `v_L10` / SAE features fire across prompts") requires Phase 3 cross-prompt M5a + M5b runs (week 3, scheduled next).
- One Phase 2 LLaVA-1.5 result (line/scene/none cell) had Phase 1 vs Phase 2 PMR Δ +0.014 — the 2nd-largest deviation across all (model × prompt × cell) tuples. Within sampling noise.
- **Lexicon expansion re-score (audit 2026-04-28)**: after `score_describe` was extended mid-Phase-3 with stems `impact`, `about to (fall|hit|impact|land|drop|bounce)`, `is about to`, `going to (fall|hit)`, the entire Phase 2 was re-summarized. Deltas are microscopic — max +0.006 on label-averaged PMR (Qwen describe 0.459 → 0.465; Idefics2 describe 0.595 → 0.597), max +0.033 on a single cell (Idefics2 textured/blank/none describe 0.000 → 0.033). All 15/15 H2 cells remain positive (+0.006 to +0.344 range unchanged); all per-(model, prompt) ladder orderings preserved. The Phase 2 headline survives the lexicon expansion. Phase 3 flip/break counts (Qwen describe M5a 10/10 / M5b 0/10; yesno 0/10 / no break) are robust to the same lexicon expansion by inspection: a *broader* physics lexicon makes "break" *strictly harder* to observe (more interventions count as physics-mode), so observing 0/10 break under the broader lexicon entails 0/10 break under the original lexicon — no Phase 3 re-scoring needed.
